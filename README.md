# Klaytn Voting Contract

## Features:

- Buy vote token
- Create proposal (fee: 20 token )    
- Vote for proposal 
- Lock vote token for 1 / 3 / 12 months to gain vote power (multiply by 1 / 3 / 12)
- Withdraw token after lock time over
